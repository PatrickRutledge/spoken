# Spoken

A Windows-first Bible app that formats Scripture into readable, justified prose with poetry preserved as line-broken blocks, small caps for the divine name, smart quotes, em dashes, and export to PDF with footer (Passage Â· Page n of X). KJV, ASV, and WEB are bundled; other public-domain translations are available via a curated catalog.

- Tech: .NET 8, .NET MAUI (Windows via WinUI 3), QuestPDF
- Core: USFM ingestion from zipped translation files, prose formatter, PDF export
- App: Minimal blue theme (#0A5BCB), passage input, translation picker, preview, export
